# Fas-LB

## 项目简介

本项目主要是实现SDN新型网络架构的的网络流量负载均衡，主要采用了SDN领域中的Floodlight控制器、Mininet仿真技术等结合Java语言开发实现，负载均衡算法采用仿生算法——蚁群算法进行路径规划
